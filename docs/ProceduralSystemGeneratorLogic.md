은하 생성 절차 (코드 없이 순서만)

1. 입력 파라미터 정의 (global seed, time)


2. seed + time → 결정론적 서브시드 파생


3. 은하 타입 샘플링 (허블 시퀀스 기반 확률 분포)


4. 총 질량  로그정규분포 샘플링
	M_gal = exp(mu + sigma * Z)
	mu: 평균
	sigma: 표준편차
	Z: 표준정규분포시드
	func get_Z(u1: float, u2: float) -> float:
		u1 = max(u1, 1e-9)
		return sqrt(-2.0 * log(u1)) * cos(2.0 * PI * u2)


5. 바리온 분율, 가스 분율 샘플링 #


6. 디스크/벌지/헤일로 질량 분해 (비율도 확률 분포) #


7. 암흑물질 헤일로 프로파일 파라미터 생성 (예: concentration 등) #


8. 디스크 스케일 길이  로그정규 샘플링


9. 디스크 두께  샘플링


10. 벌지 Sérsic index 및 반지름 샘플링


11. 강착 원반(accretion disk) 존재 여부 및 파라미터 생성


12. 중심 블랙홀 질량 샘플링 (M–sigma 관계 기반 확률화)


13. AGN 활성도 및 강착률 샘플링


14. 방출/제트 특성 파라미터화


15. 별 형성률(SFR) 로그정규 샘플링


16. 금속도 분포 생성 (반경 의존 gradient 포함)


17. 나선팔 개수 샘플링


18. 피치 각(pitch angle) 로그정규 샘플링


19. 팔 강도(contrast) 샘플링


20. 팔 위상 오프셋 생성


21. 반경 방향 표면밀도 프로파일 생성 (지수 디스크 기반)


22. 확률 밀도장(PDF) 구성 (디스크 + 벌지 + 팔 + 노이즈)


23. 별 위치 샘플링 (PDF 기반, 비균일 샘플링)


24. 최소 거리/중력적 안정성 필터 적용 (Hill / Jeans 조건)


25. 항성 질량 IMF 샘플링 (로그정규/파워법칙 혼합)


26. 항성 진화 단계 샘플링 (나이 기반)


27. 광도/온도/반지름 파생


28. 성단/성운 생성 (클러스터링 확률 기반)


29. 가스 구름 분포 생성 (난류 + 프랙탈 분포)


30. 회전 곡선 생성 (암흑물질 + 바리온 합성)


31. 각 별의 공전 속도 및 궤도 결정


32. 시간 변수 적용 → 위상(orbit phase) 계산


33. 모든 위치/속도에 시간 기반 위상 이동 적용


34. 주기성 확보를 위한 time wrapping (modulo 또는 주기 함수)


35. 동일 seed+time → 동일 상태 보장


36. 최종 은하 상태 출력 (완전 결정론적, 상태 저장 없이 재생성 가능)
